# Presentation Constraints

Keep this note next to the report so public materials stay compliant with the
challenge instructions.

## Allowed in presentation/poster

- official MNIST test accuracy
- confusion matrices on the official test set
- per-class behavior on the official test set
- comparison between in-class and extra classifiers on the official test set

## Not allowed in presentation/poster

- challenge accuracy
- tables or charts derived from challenge accuracy
- statements that reveal the final challenge ranking

## Why this file exists

The challenge README explicitly says the instructor will reveal challenge
results during the session. Keeping this note separate helps avoid accidentally
copying private numbers into public slides or poster figures.
