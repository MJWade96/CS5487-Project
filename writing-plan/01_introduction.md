# Section 1 — Introduction

The task in this project is ten-class handwritten digit classification from fixed 784-dimensional grayscale vectors. Although the benchmark is standard, it remains a meaningful test for OCR-oriented classification because models must separate visually similar classes under strong writer-style variation, including stroke thickness, slant, loop closure, and local deformation. Under this setting, robust performance indicates that a model can generalize beyond memorized appearance patterns and capture class-discriminative structure from raw pixels.

This report studies a focused question: under the fixed CS5487 protocol, how do model family and preprocessing interact in determining generalization quality? The protocol uses two official writer-disjoint trials, so training and testing writers are separated by construction. This makes the comparison a stricter evaluation of out-of-writer generalization instead of in-style fitting and provides a fair basis for comparing representation-sensitive and optimization-sensitive methods.

To answer this question fairly, we compare six model families (1-NN, logistic regression OvA, linear SVM OvA, RBF SVM OvA, Random Forest, and MLP) across multiple preprocessing pipelines using the same official trial splits. Model selection is performed using training-only cross-validation in each trial, and official-test scores are then reported on the untouched test partitions. This shared protocol isolates model and preprocessing effects while avoiding data leakage.

The headline result is clear: RBF SVM OvA achieves the best mean official-test accuracy at 0.9473, while the course baseline 1-NN reaches 0.9160, a gap of 0.0313 (summary_by_model.csv; final_selected_models.csv). Beyond ranking, the experiments show a model-dependent preprocessing pattern: min-max scaling benefits logistic regression and MLP, PCA helps linear SVM, and raw pixels remain strongest for RBF SVM, Random Forest, and 1-NN. Section 2 details the methodology and protocol, Section 3 presents the experimental setup, and Section 4 reports the quantitative and error-analysis results.
